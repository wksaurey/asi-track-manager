# asi-track-manager

## Setup Instructions
- Create Virtual Environment 
    `mkdir -p .venv/asi-track-manager`
    `python3 -m venv .venv/asi-track-manager`
- Activate Virtual Environment
    `. ./.venv/asi-track-manager/bin/activate`
- Import requirements
    `python3 -m pip install -r requirements.txt`

## Dev Instructions
- Activate venv each time you start work
- Update requirements.txt each time you install another module
    `python3 -m pip freeze > requirements.txt`
